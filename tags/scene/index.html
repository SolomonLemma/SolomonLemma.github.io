Project: Predicting Antibody-Antigen Interaction for Vaccine Development

This project was exploring the optimal predicting model for the interaction of antibody-antigens based on their proteins sequence and physicochemical properties in the use of vaccine development. XGBoost, Random Forest, Logistic Regression, and Deep Learning were employed to select the best performing models. The results showed that all the models performed more than 50% of the baseline. Specifically, random forest achieved about 90% accuracy with a threshold of 91% for keeping all the positive predictions.
